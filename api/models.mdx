## Domain Models

### BiddingParticipant

Represents participants in public procurement bidding processes.

**Namespace:** `model`  
**Table:** `BiddingParticipants`

<ParamField path="Id" type="int">
  Primary key
</ParamField>

<ParamField path="Ref" type="Guid">
  Unique reference code for participant tracking
</ParamField>

<ParamField path="NaturalPerson" type="Boolean">
  `true` for individual, `false` for legal entity
</ParamField>

<ParamField path="Name" type="string">
  Full name or company name
</ParamField>

<ParamField path="IdentificationOrNit" type="string">
  National ID or Tax ID (NIT)
</ParamField>

<ParamField path="Phone" type="string">
  Landline phone number
</ParamField>

<ParamField path="Cellular" type="string">
  Mobile phone number
</ParamField>

<ParamField path="Address" type="string">
  Physical address
</ParamField>

<ParamField path="City" type="string">
  City of residence/operation
</ParamField>

<ParamField path="Email" type="string">
  Contact email address
</ParamField>

<ParamField path="DateCreate" type="DateTime">
  Registration timestamp
</ParamField>

<ParamField path="Proposals" type="string">
  Filename of uploaded proposal PDF
</ParamField>

<ParamField path="MasterId" type="int?">
  Foreign key to Master (announcement/convocatoria)
</ParamField>

<ParamField path="Master" type="Master">
  Navigation property to related announcement
</ParamField>

**Example:**
```csharp
var participant = new BiddingParticipant
{
    Ref = Guid.NewGuid(),
    NaturalPerson = true,
    Name = "María García López",
    IdentificationOrNit = "43123456",
    Phone = "6048211234",
    Cellular = "3101234567",
    Address = "Carrera 10 #15-20",
    City = "Santa Fe de Antioquia",
    Email = "maria.garcia@example.com",
    DateCreate = DateTime.Now,
    Proposals = "propuesta-maria-garcia-2026.pdf",
    MasterId = 5
};
```

## Data Transfer Objects (DTOs)

### BiddingParticipantCreateDTO

DTO for creating new bidding participants.

**Namespace:** `modelDTOs`

<ParamField path="NaturalPerson" type="Boolean" required>
  Person type selection
</ParamField>

<ParamField path="Name" type="string" required>
  Max 150 characters
</ParamField>

<ParamField path="IdentificationOrNit" type="string" required>
  Max 20 characters
</ParamField>

<ParamField path="Phone" type="string" required>
  Landline number
</ParamField>

<ParamField path="Cellular" type="string" required>
  Max 10 characters
</ParamField>

<ParamField path="Address" type="string" required>
  Max 150 characters
</ParamField>

<ParamField path="City" type="string" required>
  Max 50 characters
</ParamField>

<ParamField path="Email" type="string" required>
  Must be valid email format
</ParamField>

<ParamField path="Proposals" type="IFormFile" required>
  PDF file, max 10MB
</ParamField>

<ParamField path="MasterId" type="int?">
  Associated announcement ID
</ParamField>

**Validations:**
- `[PdfSizes]` - Max 10MB
- `[ValidateExtensionPDF]` - Must be `.pdf`
- `[EmailAddress]` - Valid email format
- `[Required]` and `[MaxLength]` attributes on fields

**Usage:**
```csharp
var dto = new BiddingParticipantCreateDTO
{
    NaturalPerson = true,
    Name = "Juan Pérez",
    IdentificationOrNit = "1234567890",
    Phone = "6041234567",
    Cellular = "3001234567",
    Address = "Calle 10 #20-30",
    City = "Medellín",
    Email = "juan@example.com",
    Proposals = pdfFile,
    MasterId = 5
};
```

## Custom Validation Attributes

### ImageSizes

Validates image file size.

**Location:** `modelDTOs.CustomValidations`  
**Max Size:** 2MB (2,097,152 bytes)

### PdfSizes

Validates PDF file size.

**Location:** `modelDTOs.CustomValidations`  
**Max Size:** 10MB (10,485,760 bytes)

### ValidateExtensionImg

Validates image file extensions.

**Allowed:** `.jpg`, `.jpeg`, `.gif`, `.png`

### ValidateExtensionPDF

Validates PDF file extension.

**Allowed:** `.pdf`

## Relationships

### One-to-Many

<ResponseField name="Master → BiddingParticipants" type="1:N">
  One announcement can have multiple participants
</ResponseField>

<ResponseField name="Document → FileDocument" type="1:N">
  One document can have multiple file attachments
</ResponseField>

<ResponseField name="Master → FileDocument" type="1:N">
  One master entry can have multiple files
</ResponseField>

<ResponseField name="Master → Documents" type="1:N">
  One master can be associated with multiple documents
</ResponseField>

### Foreign Keys

```csharp
BiddingParticipant.MasterId → Master.Id
Document.MasterId → Master.Id
FileDocument.DocumentoId → Document.ID
```

## Database Configuration

Entity configurations are located in `src/persistenDatabase/Config/`:

- `BiddingParticipantConfig.cs`
- `CategoryConfig.cs`
- `DocumentConfig.cs`
- `EmployeeConfig.cs`
- `FileDocumentConfig.cs`
- `MasterConfig.cs`
- `PQRSDConfig.cs`
- `ProductConfig.cs`

These use Fluent API to configure:
- Primary keys
- Foreign key relationships
- Column types and constraints
- Index definitions
- Default values

## IBiddingParticipantService

Manages bidding participants in public procurement processes.

**Namespace:** `services`  
**Implementation:** `BiddingParticipantService`

### Dependencies

<ParamField path="_context" type="ApplicationDbContext">
  Entity Framework database context
</ParamField>

<ParamField path="_mapper" type="IMapper">
  AutoMapper for DTO conversions
</ParamField>

<ParamField path="_uploadedFileIIS" type="IUploadedFileIIS">
  File upload service for IIS
</ParamField>

<ParamField path="_emailSendGrid" type="IEmailSendGrid">
  SendGrid email service for notifications
</ParamField>

### Methods

#### GetAll

Retrieves all bidding participants with related Master data.

```csharp
Task<IEnumerable<BiddingParticipant>> GetAll()
```

<ResponseField name="return" type="Task<IEnumerable<BiddingParticipant>>">
  Collection of all participants with included Master relationship
</ResponseField>

**Implementation:**
```csharp
return await _context.BiddingParticipants
    .Include(x => x.Master)
    .ToListAsync();
```

#### GetById

Retrieves a participant by ID.

<ParamField path="id" type="int?" required>
  Participant identifier
</ParamField>

```csharp
Task<BiddingParticipant> GetById(int? id)
```

<ResponseField name="return" type="Task<BiddingParticipant>">
  Participant entity with Master navigation property loaded
</ResponseField>

#### GetByRef

Retrieves a participant by their unique reference GUID.

<ParamField path="reference" type="string" required>
  GUID reference code (converted internally)
</ParamField>

```csharp
Task<BiddingParticipant> GetByRef(string reference)
```

<ResponseField name="return" type="Task<BiddingParticipant>">
  Participant matching the reference code
</ResponseField>

#### Create

Registers a new bidding participant with proposal file and email notifications.

<ParamField body="model" type="BiddingParticipantCreateDTO" required>
  Participant data
</ParamField>

```csharp
Task<BiddingParticipantDTO> Create(BiddingParticipantCreateDTO model)
```

<ResponseField name="process" type="object">
  <Expandable title="creation-process">
    1. Generate unique GUID reference
    2. Upload proposal PDF to `filesBiddingParticipant` folder
    3. Create participant record in database
    4. Generate HTML email from template
    5. Send confirmation emails to:
       - Participant email
       - asesoria@espsantafedeantioquia.co
       - gerencia@espsantafedeantioquia.co
  </Expandable>
</ResponseField>

**Example:**
```csharp
var participant = await _biddingParticipantService.Create(new BiddingParticipantCreateDTO
{
    NaturalPerson = true,
    Name = "Juan Pérez",
    IdentificationOrNit = "1234567890",
    Phone = "6045551234",
    Cellular = "3001234567",
    Address = "Calle 10 #20-30",
    City = "Santa Fe de Antioquia",
    Email = "juan@example.com",
    Proposals = proposalPdfFile,
    MasterId = 5
});
```

#### DuplicaIdentificationOrNit

<ParamField path="identificationOrNit" type="string" required>

<ResponseField name="return" type="Boolean">

Removes a participant and their uploaded proposal file.

<ParamField path="id" type="int" required>

</ParamField>


## Service Layer Patterns

### Dependency Injection

All services are registered in `Startup.cs` and injected via constructor:

```csharp
services.AddScoped<IBiddingParticipantService, BiddingParticipantService>();
services.AddScoped<IBlogService, BlogService>();
services.AddScoped<ICategoryService, CategoryService>();
services.AddScoped<IDocumentService, DocumentService>();
```

### AutoMapper Usage

DTOs are mapped to entities and vice versa:

```csharp
var dto = _mapper.Map<BiddingParticipantDTO>(entity);
var entity = _mapper.Map<BiddingParticipant>(createDto);
```

### Error Handling

Services with complex operations use transactions:

```csharp
using var transaction = _context.Database.BeginTransaction();
try
{
    // Operations
    transaction.Commit();
}
catch (Exception ex)
{
    transaction.Rollback();
}
```

### Async/Await

All database operations are asynchronous for better performance.

#### DeleteConfirmed

Removes announcement with complete cascade deletion using transaction.

<ParamField path="id" type="int" required>
  Announcement ID to delete
</ParamField>

```csharp
Task DeleteConfirmed(int id)
```

<ResponseField name="cascade-deletion" type="object">
  <Expandable title="deletion-steps">
    1. Begin transaction
    2. Load Master with Documents and FileDocument collections
    3. Delete cover page file from `nacionLicitante` folder
    4. For each associated Document:
       - Load its FileDocument collection
       - Delete each physical file from `filesDocuments`
       - Remove FileDocument records
       - Remove Document record
    5. For each FileDocument linked directly to Master:
       - Delete physical file
       - Remove FileDocument record
    6. Remove Master record
    7. Commit transaction
    8. Rollback on any exception
  </Expandable>
</ResponseField>

#### Create

Creates new PQRSD submission and sends email notifications.

<ParamField body="model" type="PQRSDCreateDto" required>
  PQRSD data
</ParamField>

```csharp
Task<PQRSDDto> Create(PQRSDCreateDto model)
```

<ResponseField name="process" type="object">
  <Expandable title="creation-and-notification">
    1. Generate unique Guid for PQRSDID
    2. Set creation timestamp
    3. Create PQRSD entity with:
       - NamePerson
       - Email
       - PQRSDName (subject)
       - Description
       - NameSotypeOfRequest (type)
       - IsAnswered = false
    4. Save to database
    5. Load HTML email template from `/emailTemplete/PQRSDEmail.html`
    6. Replace placeholders: {PQRSD}, {Name}, {Email}, {Asunto}, {Description}
    7. Send emails to:
       - Citizen email address
       - asesoria@espsantafedeantioquia.co
       - admin@espsantafedeantioquia.co
  </Expandable>
</ResponseField>

**Example:**
```csharp
var pqrsd = await _pqrsdService.Create(new PQRSDCreateDto
{
    NamePerson = "Carlos Ramírez",
    Email = "carlos@example.com",
    PQRSDName = "Solicitud de instalación de servicio",
    Description = "Requiero instalación...",
    NameSotypeOfRequest = "Petición"
});
// Returns PQRSDID (Guid) for tracking
```

# Welcome to ESP Santa Fe de Antioquia

ESP Santa Fe de Antioquia is a comprehensive web-based management system designed for the Empresa de Servicios Públicos (Public Services Company) of Santa Fe de Antioquia, Colombia. Built on ASP.NET Core 3.1, this platform streamlines municipal service operations and citizen interactions.

## What is ESP Santa Fe de Antioquia?

This system serves as the digital backbone for managing public services, providing both administrative tools and citizen-facing features. It enables efficient management of service requests, document workflows, employee coordination, and public communications.

## Technology Stack

The platform is built with modern web technologies:

- **Framework**: ASP.NET Core 3.1 MVC
- **Database**: SQL Server with Entity Framework Core
- **Authentication**: ASP.NET Core Identity
- **Email**: SendGrid integration
- **UI**: Razor Pages with responsive design
